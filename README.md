### 基于 Create React App 脚手架开发 

1. 通过 **react-app-rewired** 重写webpack配置文件


### 目录结构

```bash
./
├── README.md
├── package.json
├── src                             # 源码目录
    ├── pages                         # 页面
    ├── components                    # 公用组件
    ├── hooks                         # 自定义hooks
    ├── styles                        # 全局样式
    ├── assets                        # 静态资源
    ├── router                        # 全局页面路由
    ├── router                        # 请求方法、公用方法等
├── config-overrides.js             # 重写webpack配置文件

```
